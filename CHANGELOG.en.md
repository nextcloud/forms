# Changelog

This file documents the recent changes visible to end users of the Forms app.

## v4.2.1

Welcome to the new version of your favorite Forms app ✨

* You can now embed forms into websites 🌐
* Forms can now be archived 📦 for better and cleaner overview.
* Besides having a expiration date, Forms now can also be closed manually 🔐
* We’re now using custom dialogs to show you warnings instead of the ugly browser ones wherever possible 💬
* You can now link a file within your Nextcloud to the submissions of a form.
  With this comes also a greater choice for the file format:
  Besides the known CSV export, we now support ODS and XLSX files, too! 🎉
* You can now clone an existing question and create a new one from it.

Besides those new features ✨ we also included many bug fixes 🐛, updated translations 🗣 and dependencies.

Feel free to help improving it by reporting any issues you encounter, help translating it on Transifex or contribute on [GitHub](https://github.com/nextcloud/forms).
